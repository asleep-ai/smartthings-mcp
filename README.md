# SmartThings MCP Server

A Model Context Protocol (MCP) server for interacting with the SmartThings API.

## Installation

```bash
uv pip install -e .
```

## Configuration

The server requires a SmartThings Personal Access Token. You can generate one from the [SmartThings Developer Portal](https://account.smartthings.com/tokens).
